GPA/CGPA Calculator 

A simple web-based GPA and CGPA calculator built using HTML, CSS, and JavaScript.  
It allows students to:
-add subjects
-calculate semester GPA
-running CGPA
-simulate future semesters using a What-If mode.

Features
-Add subjects with credits and grades
-Automatic GPA calculation
-Running CGPA calculation
-What-If mode for future semesters (up to 8 semesters)
-Subjects due counter

How to RUN:
1.Download or clone the repository
2.Open "index.html" in a browser
3.Start adding subjects and view results

Note:
No frameworks used. Pure frontend project.
